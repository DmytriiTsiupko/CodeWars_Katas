#Armstrong number is a positive number which is the sum of if its own digits, each raised to the power of the power of
#the given base. In this Kata, we will ourselves to decmal (base 10).

          ######################################   FOR EXAMPLE   ######################################

Take 153 (3 digits), which is Armstrong number:

1^3 + 5^3 + 3^3 = 1 + 125 + 27 = 1533

and 1654 (4 digits), witch isn't:

1^4 + 6^4 + 5^4 + 2^4 = 1 + 1296 + 625 + 16 = 1938

###########     The Challeng     ###########

Your code must return True or False (not 'true' and 'false') depending upon whether the given number is a Armstrong
number in base 10. This may be True or False in your language, e.g.PHP.
